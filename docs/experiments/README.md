# Experiment Log

Chronological reports on render sessions, parameter sweeps, and stylistic explorations.

## Template

```markdown
# Experiment: YYYY-MM-DD_Title

## Goal
- Brief objective or hypothesis.

## Inputs
- Prompt(s)
- Models / checkpoints
- Sampler, steps, CFG scale
- Ancillary assets (textures, palettes, etc.)

## Process
- Iteration notes, parameter tweaks, decisions made.

## Results
- Thumbnails stored in `assets/previews/`
- Links to full-resolution outputs (external storage if large)

## Learnings
- Key takeaways, follow-up ideas, or blockers.
```

## Contribution Notes

- Commit experiments as separate files: `YYYY-MM-DD_title.md`.
- Reference related assets or workflows with relative paths.
- Capture both wins and failures to preserve context.
